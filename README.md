# QDII 美股基金额度助手

实时查看国内美股 QDII 基金的申购额度状态（开放申购 / 限大额 / 暂停申购），并展示人民币、美元份额的单日申购限额。

## 2.0 版本更新

- 新增美元份额额度展示，例如 `10美元`、`400美元`、`1000美元`
- 新增基金公告解析：从天天基金【基金公告】里的【基金销售】公告抓取限额
- 新增直销渠道限额识别：公告写明“直销渠道/直销机构”时，会单独展示直销额度
- 新增多渠道限额明细：支持同时展示 `直销`、`代销`、`天天基金`、`公告` 等来源
- 优化单日限额展示布局：详情区用标签展示多条限额，避免挤在一行
- 修复直接双击 `index.html` 导致 `data.json` 无法加载的问题提示

## 功能

- 打开页面自动加载全部 QDII 基金数据（700+ 只）
- 默认展示热门美股相关基金（纳指100、标普500 等）
- 按申购状态分组：可买入 / 限大额 / 暂停申购
- 展示具体单日限额，支持人民币份额、美元现汇/现钞份额
- 展示多渠道限额明细，如 `直销 10万元 / 天天基金 100元`
- 支持搜索任意基金代码或名称，添加到"自选"
- 自选列表保存在浏览器本地（localStorage），下次打开自动恢复
- 手机、电脑均可使用（响应式布局）
- 自动适配深色模式

## 使用方式

### 本地 HTTP 服务

```bash
cd /Users/wangjinlan/Downloads/qdii-monitor-main
python3 -m http.server 8899
```

然后访问 http://localhost:8899

> 不建议直接双击 `index.html` 打开。现代浏览器会限制 `file://` 页面读取同目录下的 `data.json` 和 `market.json`，导致基金列表无法加载。

### 部署到 Gitee / GitHub Pages

1. 创建一个新的 Git 仓库
2. 将 `index.html` 推送到仓库
3. 在仓库设置中开启 Pages 服务
4. 得到公网链接，可分享给他人

## 数据来源

数据来自天天基金（东方财富）公开接口和基金公告。

- 基金列表与申购状态：`fundmobapi.eastmoney.com`
- 基金销售公告列表：`api.fund.eastmoney.com/f10/JJGG`
- 公告正文/PDF：用于解析大额申购、直销渠道、美元份额等限额
- 市场行情：Yahoo Finance Chart API
- 生成文件：`data.json`、`market.json`

## 注意事项

- 数据仅供参考，实际申购限额以基金公司最新公告为准
- 天天基金接口非官方公开 API，请勿高频请求
- 不同渠道限额可能不同，页面会尽量展示 `直销`、`代销`、`天天基金` 等明细
- 如接口或公告格式变更导致无法加载，需更新 `scripts/fetch_data.py` 中的抓取和解析逻辑

## 技术栈

- 纯 HTML + CSS + JavaScript，单文件，零依赖
- Python 数据抓取脚本
- 天天基金 REST API + 基金销售公告解析
- localStorage 持久化用户自选
- GitHub Actions 定时更新数据并部署 GitHub Pages
